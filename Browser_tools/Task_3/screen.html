Screen

  • availHeight - Величина высоты монитора пользователя в пикселях за вычетом интерфейса виндоус.
  • availWidth - Величина ширины монитора пользователя в пикселях за вычетом интерфейса виндоус.
  • colorDepth - Величина глубины цветовой палитры для отображения изображений в битах на пиксель.
  • height - Величина высоты монитора пользователя в пикселях.
  • pixelDepth - Величина цветнового разрешения (в битах на пиксель) экрана пользователя.
  • Width - Величина высоты монитора пользователя в пикселях.
